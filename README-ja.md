# Tsumiki.vst3

微分音対応バーチャルアナログ/FMシンセサイザー

[English](./README.md)

VST3 プラグインが動作する DAW で使用できます。  
プリセットを利用するには、 `.vstpreset` ファイルに対応した DAW が必要です。
