================================================================================
Tsumiki
================================================================================

【インストール】
install.ps1 を実行してください。
(右クリック → PowerShell で実行)

DAW で読み込みがブロックされてしまう場合、 Minori-Tsumiki.cer をダブルクリックして「証明書のインストール」を行ってください。

【使い方】
https://github.com/mino-ri/Tsumiki/blob/main/Documents/README.md

【アンインストール】
以下のファイル・フォルダを削除してください。
C:\Program Files\Common Files\VST3\Tsumiki.vst3
C:\ProgramData\VST3 Presets\Minori\Tsumiki\

================================================================================

[Installation]
Please run install.ps1.
(Right-click → Run with PowerShell)

If your DAW blocks loading the plugin, double-click Minori-Tsumiki.cer and install the certificate.

[Usage]
https://github.com/mino-ri/Tsumiki/blob/main/Documents/README.md

[Uninstallation]
Please delete the following files and folders:
C:\Program Files\Common Files\VST3\Tsumiki.vst3
C:\ProgramData\VST3 Presets\Minori\Tsumiki\
